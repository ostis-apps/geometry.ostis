Задачей scp-программы применения аналогии на бинарных неориентированных отношениях является применение аналогии на бинарных неориентированных отношениях для поиска значения неизвестной величины. Первым входным параметром данной scp-программы является величина, значение которой нужно найти или число, для которого нужной найти идентификатор; вторым -- геометрическая фигура, для которой необходимо найти неизвестную величину; третьим -- объект базы знаний, связанный с объектом, на котором задана неизвестная величина, отношением аналогии; четвёртым -- знак отношения, которым связан объект, на котором задана неизвестная величина с самой неизвестной величиной; с пятого по восьмой -- sc-действия дерева декомпозиции решения; девятым -- сообщение об успешном или безуспешном проведении вывода по аналогии на бинарных неориентированных отношениях; десятым -- знак отношения аналогии. В ходе выполнения программы явно указывается связь со следующими ключевыми узлами: отношениями периметр*, площадь* и длина*, уточняющими связки отношений, в которые входит неизвестная величина; отношением равенство величин*, с помощью которого связывается параметр неизвестного объекта со значением соответствующего параметра известного объекта базы знаний; отношением значение*, использующимся для нахождения в базе знаний объектов с известным значением.
