Задачей агента поиска всех утверждений и определений, ассоциированных с заданной сущностью является поиск всех утверждений и определений, ассоциированных с заданной сущностью. Т.е. поиск всех утверждений и определений, непосредственно связанных с заданной сущностью и описывающих сущности, с которыми заданная сущность связанна с помощью отношений включение* и разбиение*. Данный агент инициируется при условии появления в памяти вопросной конструкции, соответствующей запросу вывода всех утверждений и определений, ассоциированных с заданной сущностью.
