Команда пользовательского интерфейса для генерации противоположной теоремы предназначена для генерации противоположной теоремы. Единственным аргументом команды является теорема, для которой необходимо сгенерировать противоположную теорему. Результатом выполнения команды является вывод на экран исходной теоремы и противоположную ей.
