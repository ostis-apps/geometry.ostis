Команда поиска семантических окрестностей, у которых не указан ключевой sc-элемент предназначена для нахождения емантических окрестностей, у которых не указан ключевой sc-элемент. Единственным аргументом команды является структура включающая семантические окрестности. Результатом выполнения команды является является структура в которой необходимо найти семантические окрестности, у которых не указан ключевой sc-элемент.
