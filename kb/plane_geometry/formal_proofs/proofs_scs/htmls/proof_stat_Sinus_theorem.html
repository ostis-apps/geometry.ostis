1. Пусть АВС - треугольник со сторонами a, b, c и противолежащими углами α, β, γ. Докажем, что a / sinα = b / sinβ = c / sinγ.
  2. Опустим из вершины С высоту CD.
  3. Из прямоугольного треугольника ACD:
    • если ∠ α острый, получаем CD = b · sinα.
    • если ∠ α тупой, то CD = b · sin(180- α) = b · sinα
  4. Аналогично из треугольника BCD получаем CD = a · sinβ.
  5. Итак, a · sinβ = b · sinα.
  6. Отсюда a / sinα = b / sinβ.
  7. Аналогично доказываем b / sinβ = c / sinγ.
