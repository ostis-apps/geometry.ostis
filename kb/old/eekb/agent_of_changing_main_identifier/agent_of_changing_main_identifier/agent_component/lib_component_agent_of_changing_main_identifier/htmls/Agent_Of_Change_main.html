Задачей агент изменения основного идентификатора заданного sc-элемента на указанный файл является удаление старого основного идентификатораи установка нового, выбранного пользователем. Данный агент инициируется при условии появления в памяти вопросной конструкции, соответствующей запросу замены основного идентификатора. Первым аргументом запроса является узел, в котором требуется заменить основной идентификатор. Вторым аргументом запроса является файл на который требуется заменить текущий узел.
