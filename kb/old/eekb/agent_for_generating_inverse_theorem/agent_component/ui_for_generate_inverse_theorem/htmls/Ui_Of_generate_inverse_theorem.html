Команда пользовательского интерфейса для генерации обратной теоремы предназначена для генерации обратной теоремы. Единственным аргументом команды является теорема, для которой необходимо сгенерировать обратную теорему. Результатом выполнения команды является вывод на экран исходной теоремы и обратную ей.
