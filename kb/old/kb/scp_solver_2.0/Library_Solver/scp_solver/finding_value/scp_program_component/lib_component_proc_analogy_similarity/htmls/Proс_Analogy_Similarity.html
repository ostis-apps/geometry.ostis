Задачей scp-программы применения аналогии на бинарных ориентированных отношениях является применение аналогии на бинарных ориентированных отношениях для поиска значения неизвестной величины. Первым входным параметром данной scp-программы является величина, значение которой нужно найти или число, для которого нужной найти идентификатор; вторым -- геометрическая фигура, для которой необходимо найти неизвестную величину; третьим -- объект базы знаний, связанный с объектом, на котором задана неизвестная величина, отношением аналогии; четвёртым -- знак отношения, которым связан объект, на котором задана неизвестная величина с самой неизвестной величиной; с пятого по восьмой -- sc-действия дерева декомпозиции решения; девятым -- сообщение об успешном или безуспешном проведении вывода по аналогии на бинарных ориентированных отношениях; десятым -- знак отношения аналогии. В ходе выполнения программы явно указывается связь со следующими ключевыми узлами: порядковыми атрибутами 1' и 2', уточняющими вхождение элементов в связку отношения аналогии; отношениями периметр*, площадь* и длина*, уточняющими связки отношений, в которые входит неизвестная величина; отношениями возведение в степень* и произведение величин*, ролевыми отношениями показатель степени' и основание степени', использующимся для вычисления значения неизвестной величины; числом 2, использующимся для возведения в степень* в качестве показателя степени;
