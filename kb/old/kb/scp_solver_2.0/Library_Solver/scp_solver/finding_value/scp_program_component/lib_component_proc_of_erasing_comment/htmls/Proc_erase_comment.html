Задачей scp-программы удаления комментария является удаление комментария, поясняющего узел вопроса. Единственным входным параметром является узел вопроса, для которого происходит удаление комментария. В ходе выполнения программы явно указывается связь со следующими ключевыми узлами: абсолютными понятиями sc-комментарий и английский язык, неролевым отношением трансляция sc-текста* и ролевыми отношениями ключевой sc-элемент' и пример', использующимися для установления узла комментария, который необходимо удалить.
