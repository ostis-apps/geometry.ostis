Команда поиска неизвестной величины предназначена для нахождения значения неизвестной величины или идентификации числа. Единственным аргументом команды является неизвестная величина, для которой необходимо найти значение, или число, для которого нужно найти идентификатор. Результатом выполнения запроса является появление в памяти цепочки значения искомой величины или цепочки идентификации искомого числа.
