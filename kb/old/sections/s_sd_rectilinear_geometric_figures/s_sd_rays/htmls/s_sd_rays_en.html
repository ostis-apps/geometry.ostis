Definition. Ray - is a part of straight line, which consist of all points of the straight line, which lying on one side of the point, which called initial point, including this point.

Designation. Ray is set by initial point and by direction or by initial point and another (any) it’s own point. AB - is denotion of the ray with initial point A, including point B.

Types of rays:

  • Additional rays is called different ray of one straight line which have common initial point.
  • Half-straight lines is called opened ray that is a ray from which initial point is excluded.

  • The same direction, or codirectional, called two rays (or two half-straight lines), if it’s possible to combine them by parallel transfer.

    If rays a and b have the same direction and rays b and c also have the same direction, then a and c have the same direction.
  • Opposite directed rays is called two rays (or two half-straight lines), if each of them have the same direction with the ray (or two half-straight lines), an additional ray to another.
