Задачей агента генерации обратной теоремы является генерация обратной теоремы. На вход данному агенту подается теорема, для которой необходимо сгенерировать обратную теорему.Данный агент инициируется при условии появления в памяти вопросной конструкции, соответствующей запросу генерации обратной теоремы.Результатом выполнения команды является вывод на экран исходной теоремы и обратную ей.
