Задачей scp-программы добавления комментария является добавление комментария об успешности нахождения неизвестной величины или идентификации числа к узлу вопроса. Первым входным параметром данной scp-программы является узел вопроса, вторым -- текст добавляемого комментария, третьим -- множество, в которое будет занесён результат выполнения scp-программы. В ходе выполнения программы явно указывается связь со следующими ключевыми узлами: абсолютными понятиями sc-комментарий и английский язык неролевым отношением трансляция sc-текста* и ролевыми отношениями ключевой sc-элемент' и пример', необходимыми для построения установленной конструкции комментария к узлу вопроса.
